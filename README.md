## 목표

스프링 부트로 웹을 생성하고 Mariadb를 통해서 데이터베이스를 연결하고 깃을 통해 형상관리를 하는 연습.<br/>
db를 접속할때 Jasypt를 통해 프로퍼티의 암호화와 복호화를 하여 암호의 하드코딩을 방지함.<br/>
Thymeleaf 템플릿을 통해 /members로 들어가면 DB를 조회할수있도록함.<br/>
member 엔티티를 통해 데이터를 모델링함.<br/>
